connect()

Make the connection to the server. Calls `setConnector` to establish the node client connection. Calls `listen` if connection was successful, and will resolve the promise. Will retry connection on the `connectionTimeout` interval. Number of connection retries is based on `remainingRetries` Will reject the promise if connect or re-connect attempts fail.
